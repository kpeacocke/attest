# Support

Attest is an open source project.

## Getting help

- Use **GitHub Discussions** for questions, ideas, and community support.
- Use **GitHub Issues** for confirmed bugs and feature requests.
- Check existing Issues and Discussions before creating new ones.

## What to include

- Attest version (or commit SHA)
- Operating system and Python version
- How you ran it (CLI vs automation platform)
- Logs/output (redact secrets)
- Steps to reproduce (for bugs)

## Tooling

This project uses:

- **GitHub Issues** for issue tracking
- **GitHub Projects** for milestone and roadmap tracking
- **GitHub Actions** for CI/CD
- **Snyk** for security scanning
- **SonarQube** for code quality

We do not use JIRA or other external project management tools.
