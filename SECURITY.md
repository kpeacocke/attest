# Security Policy

## Supported versions

This project is in early development. Security fixes will be released on the latest minor release line.

## Security tooling

Attest uses:

- **Snyk** for automated security scanning of dependencies and first-party code
- **GitHub Security Advisories** for vulnerability disclosure and coordination
- **GitHub Dependabot** for automated dependency updates

## Reporting a vulnerability

Please **do not** file public issues for suspected vulnerabilities.

Instead, use **GitHub Security Advisories** (preferred) or contact the maintainer privately.
Include:

- a description of the issue
- steps to reproduce
- affected versions/commits
- any suggested mitigations

## Disclosure

We will aim to acknowledge receipt within a reasonable timeframe and coordinate a responsible disclosure.
